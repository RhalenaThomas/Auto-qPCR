Options

Enter endogenous control gene/chromosome(s): (seperate genes/chromosomes with spaces)

Endogenous control genes: These are the housekeeping or reference genes you wish to normalize agains. Actin or GADPH are common options. You must enter the gene names exactly as they appear in your data spread sheet. You many enter multiple control genes. Separate the genes with commas. The average of the techinical replicates will be used to calculate expression values.


Cut-Off:

Techical replicate for a given sample will be removed if the standard deviation (SD) of those replicates is about 0.3 (or the entered value). Replicates will be removed until the SD is below the cuttoff.



Max Outliers:

How many technical replicates can be removed to reach the standard deviation cut off about? The default of 0.5 means that in 3 technical replicates only one can be removed.



Enter target order: (seperate target names with spaces)

Enter the order you would like genes/chromosomes to appear on the plots.


Enter sample order: (seperate sample names with spaces)

Enter the order you would like samples to appear on the plots.


Control Sample:

The sample used as calibrator (compare with this sample) for the relative (delta delta CT) and for the stability models.
